# 🌦️ Weather App (Geolocation + City Search)

This is a responsive **Weather Application** built using **HTML, CSS,
and JavaScript** that displays real‑time weather information using the
**OpenWeatherMap API**.

Users can: - View weather using their current location - Search weather
by city name - See temperature, humidity, wind speed, and cloud
percentage

------------------------------------------------------------------------

## 🚀 Features

✔ Current location weather detection (Geolocation API)\
✔ City-based weather search\
✔ Real-time weather updates\
✔ Country flag display\
✔ Loading animation while fetching data\
✔ Tab-based UI navigation\
✔ Session storage support\
✔ Clean responsive interface

------------------------------------------------------------------------

## 🛠 Tech Stack

-   HTML5
-   CSS3
-   JavaScript (Vanilla JS)
-   OpenWeatherMap API
-   Geolocation API
-   Session Storage

## 🌍 How It Works

### Your Weather Tab

-   Click "Grant Access"
-   Browser requests location permission
-   Weather loads automatically

### Search Weather Tab

-   Enter city name
-   Press search icon
-   Weather appears instantly
